# Трекер тренировок и дел — GitHub Pages / PWA

Это готовая версия веб-приложения для GitHub Pages.

Приложение работает как PWA: открывается по ссылке, на телефоне его можно добавить на главный экран как обычное приложение.

## Что внутри

```text
index.html
manifest.webmanifest
sw.js
404.html
.nojekyll
icons/
  icon-180.png
  icon-192.png
  icon-512.png
README.md
```

## Что умеет приложение

- календарь тренировок
- отметка силовой
- отметка кардио
- добавление дел на день
- заметки по дню
- прогресс по табам: день / неделя / месяц / весь срок
- горизонтальная линия прогресса
- статистика по тренировкам и задачам
- экспорт / импорт JSON
- сохранение данных в браузере через localStorage
- PWA-режим для телефона

## Как залить через сайт GitHub

### 1. Создай репозиторий

1. Зайди на GitHub.
2. Нажми `+` → `New repository`.
3. Назови репозиторий, например:

```text
training-tracker
```

4. Выбери `Public`.
5. Нажми `Create repository`.

### 2. Загрузи файлы

Важно: загружать нужно не сам zip, а файлы из распакованной папки.

1. Распакуй архив.
2. Открой созданный репозиторий.
3. Нажми `Add file` → `Upload files`.
4. Перетащи все файлы и папку `icons` в окно загрузки.
5. Внизу нажми `Commit changes`.

В корне репозитория должен лежать файл:

```text
index.html
```

Не внутри дополнительной папки, а именно в корне.

### 3. Включи GitHub Pages

1. Открой репозиторий.
2. Перейди в `Settings`.
3. Найди раздел `Pages`.
4. В блоке `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - Folder: `/root`
5. Нажми `Save`.

Через минуту-другую GitHub даст ссылку вида:

```text
https://ТВОЙ-НИК.github.io/training-tracker/
```

Если ссылка сразу не открылась — подожди немного и обнови страницу. GitHub Pages иногда думает дольше, чем человек перед покупкой абонемента в зал.

## Как залить через Terminal

Если хочешь через Terminal:

```bash
cd путь/к/папке/training_tracker_github_pages

git init
git add .
git commit -m "Add training tracker PWA"
git branch -M main
git remote add origin https://github.com/ТВОЙ-НИК/training-tracker.git
git push -u origin main
```

Потом включи GitHub Pages в настройках репозитория:

```text
Settings → Pages → Deploy from a branch → main → /root → Save
```

## Как открыть на телефоне

После публикации открой ссылку GitHub Pages на телефоне.

### iPhone

1. Открой ссылку в Safari.
2. Нажми кнопку `Поделиться`.
3. Выбери `На экран Домой`.
4. Нажми `Добавить`.

После этого появится иконка приложения.

### Android

1. Открой ссылку в Chrome.
2. Нажми меню `⋮`.
3. Выбери `Добавить на главный экран` или `Установить приложение`.
4. Подтверди.

## Важно про данные

Данные сохраняются в браузере через `localStorage`.

Это значит:

- если открыла приложение на iPhone — данные будут в Safari на iPhone;
- если открыла на Mac — данные будут отдельно в браузере на Mac;
- чтобы перенести данные, используй `Экспорт JSON` и потом `Импорт`.

## Если приложение не обновляется после правок

PWA может кэшировать старую версию.

Что сделать:

1. Открой приложение.
2. Обнови страницу 2–3 раза.
3. Если не помогло — очисти данные сайта в браузере.
4. Либо измени имя кэша в `sw.js`:

```js
const CACHE_NAME = "training-tracker-pwa-v2";
```

Например, поменяй `v1` на `v2`, потом снова загрузи файл на GitHub.
