# Трекер тренировок — сверстанная PWA-версия

Это готовая верстка приложения по мобильному макету.

## Файлы

```text
index.html
manifest.webmanifest
sw.js
404.html
.nojekyll
icons/
README.md
```

## Как обновить текущий GitHub Pages

1. Распакуй архив.
2. Открой репозиторий `training-tracker`.
3. Удали старые файлы или просто загрузи новые с заменой.
4. Важно: `index.html` должен лежать в корне репозитория.
5. Нажми `Commit changes`.
6. Подожди зелёную галочку во вкладке `Actions`.
7. Открой ссылку GitHub Pages.

## Что реализовано

- мобильный UI по макету
- табы прогресса: День / Неделя / Месяц / Весь срок
- горизонтальный прогресс-бар
- цели: Силовая / Кардио / пользовательские цели
- серия по неделе
- дела на выбранный день
- календарь месяца
- выбор даты через календарь
- настройки
- экспорт / импорт JSON
- сброс данных
- PWA manifest
- service worker
- иконки приложения

## Данные

Данные сохраняются в браузере через localStorage.

Если открыть приложение на другом устройстве — данные будут отдельные.
Для переноса используй экспорт / импорт JSON.
