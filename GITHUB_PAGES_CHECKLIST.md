# Быстрый чеклист загрузки на GitHub Pages

1. Распаковать архив.
2. Создать Public-репозиторий на GitHub.
3. Загрузить содержимое папки в корень репозитория.
4. Проверить, что `index.html` лежит в корне.
5. Settings → Pages.
6. Source: Deploy from a branch.
7. Branch: main.
8. Folder: /root.
9. Save.
10. Открыть ссылку `https://username.github.io/repository-name/`.
11. На телефоне добавить на экран Домой.
